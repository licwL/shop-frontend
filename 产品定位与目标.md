# 产品定位与目标

## 我们做什么

企业级 B2C 电商平台，覆盖**美妆护肤、服装服饰、消费电子**三大核心品类。面向 C 端消费者与平台运营方双端角色，提供从商品浏览、交易下单到运营管理的全链路电商能力。

品牌调性：现代、简约、品质感。

## 平台构成

| 端 | 用户 | 用途 |
|----|------|------|
| 管理端 /admin | 平台运营人员 | 管理商品、订单、用户、通知 |
| 用户端 /user | 消费者 | 浏览商品、加购物车、下单、收藏、收货地址 |

**当前任务：先开发管理端，再开发用户端。**

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue 3 + Vite + Element Plus + SCSS |
| 后端 | Spring Boot 2.7 + MyBatis + JWT + MySQL（已开发完成） |
| 接口文档 | `f:/Commercial/sky-take-out/docs/API接口文档-管理端.docx` + `用户端.docx` |
| 公共代码 | `shared/request.js`（axios）、`shared/utils.js`、`shared/styles/variables.scss` |

## 核心流程

**用户购物路径**：浏览商品 → 搜索 → 详情 → 加购物车 → 下单（扣库存）→ 支付 → 商家发货 → 完成

**运营管理路径**：管理商品/分类 → 发布通知 → 处理订单 → 导出报表

## 开发原则

- Element Plus 组件优先，不造轮子
- SCSS 变量统一用 `shared/styles/variables.scss`
- axios 请求用 `shared/request.js`，不发绝对路径（Vite 代理到后端 8080）
- 先完成功能再美化，核心逻辑优先

## 后端环境

- 地址 `http://localhost:8080`，Vite 代理转发
- 鉴权：登录返回 token，请求头 `token` 字段携带
- 响应：`{ code: 1, data: {...} }` 成功，`{ code: 0, msg: "..." }` 失败
- 分页：参数 `page` + `pageSize`，返回 `{ total, records }`
