# 青蔬市集 (Fresh Market) — 产品定位与开发目标

## 我们做什么

**青蔬市集**是一个现代精品生鲜电商平台。面向注重生活品质的城市消费者，销售新鲜食材、精选农产品。品牌调性：明亮、洁净、有生机。

## 平台构成

```
青蔬市集 = 管理端（商家后台）+ 用户端（购物商城）+ 后端 API
```

| 端 | 用户 | 用途 |
|----|------|------|
| 管理端 /admin | 商家/运营人员 | 管理商品、订单、用户、通知、看数据 |
| 用户端 /user | 消费者 | 浏览商品、加购物车、下单、收藏、收货地址 |

**先开发管理端，再开发用户端。** 两端共享 `shared/` 下的 axios 封装、主题变量、工具函数。

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue 3 + Vite + Element Plus + SCSS |
| 后端 | Spring Boot 2.7 + MyBatis + JWT + MySQL |
| 接口文档 | `docs/API接口文档-管理端.docx` 和 `用户端.docx`（后端仓库） |
| 公共代码 | `shared/request.js`（axios封装）、`shared/utils.js`、`shared/styles/variables.scss` |

## 主题设计

绿色主调，明亮洁净。主色 `#6B9080`，米纸底 `#F9F6F0`。见 `shared/styles/variables.scss`。整体气质参考精品生活电商，清新自然，别做成杂货铺风格。

## 管理端（当前开发目标）

商家日常运营的工具。功能模块：

| 模块 | 说明 |
|------|------|
| 登录 | token 存储，自动带请求头，401 跳登录 |
| 仪表盘 | 营业额、订单统计、销量排行、用户增长（折线图/柱状图） |
| 商品管理 | 列表（搜索+分类筛选）、新增/编辑、批量删除、库存 |
| 订单管理 | 列表（按状态/时间筛选）、详情、发货、完成 |
| 用户管理 | 用户列表（搜索）、启用/禁用 |
| 通知管理 | 通知列表、新增/编辑、发布/撤回 |
| 导出 | 订单 Excel、商品 Excel 下载 |

侧边栏导航，内容区展示各模块页面，整体整洁专业。

## 用户端（后续开发，本次不涉及）

消费者购物流程：首页商品浏览 → 搜索 → 详情 → 加购物车 → 下单 → 支付 → 订单追踪。含收货地址管理、商品收藏、通知查看。

## 开发原则

- Element Plus 组件优先，不要自己造轮子
- SCSS 变量统一使用 `shared/styles/variables.scss`，不要写魔术数字
- axios 请求用 `shared/request.js` 封装，不直接写 `axios.get`
- API 调用发相对路径（Vite 代理到后端 8080）
- 页面功能完整再美化，核心逻辑优先

## 后端环境

- 地址：`http://localhost:8080`
- 鉴权：登录后返回 token，请求头 `token` 字段携带
- 统一响应：`{ code: 1, msg: null, data: {...} }` 表示成功
- 分页：参数 `page` + `pageSize`，返回 `{ total, records }`
